公會說明

一.建立公會條件：

  1. 只有人類王族職業才可創建公會。
  2. 等級須達10級以上，並準備10000Cp。
  3. 與查斯泰爾鎮的NPC【公會管理員(846.469)】對話，點選「建立公會」按鈕。
  4. 輸入欲創建的公會名稱，並經確認非重複之公會名稱後，即可完成公會創立。
    PS: 人類王族無法加入其他公會。

二.職位和權力：

  1. 會長：加入成員批准、踢出會員、賦予職位、賦予名稱、編輯公告、申請加入聯盟
  2. 副會長：加入成員批准、踢出會員、賦予職位、賦予名稱
  3. 長老：地位高於一般會員
  4. 一般會員

三.公會標誌：

  1. 公會標誌只能使用 24x20 BMP格式的圖片。
  2. 在公會視窗中點選「公會標誌」按鈕，選擇想要替換的公會標誌圖片路徑，並點選「套用」即可。

四.開發公會技能：

  1. 公會成員可開發公會技能，在技能持續時間內所有成員皆可獲得BUFF效果。
  2. 消耗一定素材後，即可開始開發公會技能。一位成員只能開發一項技能(不可重複)，且全公會同時間內只能開發一項技能。
  3. 開發完後的公會技能效果可維持七天，時間結束後需要點選「維持技能」按鈕，消耗一定素材後再次開啟公會技能效果。

五.公會指令與快捷：

/公會 彈出公會視窗。快捷鍵(G)
/賦予稱號 (會員名稱) (稱號) 利用此命令語賦予公會成員稱號。
/踢出公會 (會員名稱) 利用此命令語可以踢出公會成員。
/退出公會 退出加入的公會時使用。
/解散公會 公會會長解散公會時使用。必須沒有任何公會會員以及沒有佔有城鎮。

加入公會聯盟

  1. 加入公會聯盟需要一個公會的會長找到希望聯盟的另一個公會會長。
  2. 點擊[Shift+鼠標左鍵]彈出角色選單後選擇[聯盟加入申請]。
  3. 收到加入申請的一方會成為聯盟領袖公會，申請加入的公會為一般公會。
  4. 聯盟公會會員可以將聯盟領袖公會的顯示在角色名稱的左側。
  5. 自己公會的標誌顯示在名稱右側。
  6. 聯盟領袖公會會員只可顯示所屬公會的標誌。
  7. 包括所屬公會，聯盟公會最多只能加入10個。
  8. 聯盟是由數個公會集結而成的組織，可共同攻打或防守城堡。若聯盟中有公會佔領城堡，其他聯盟可享有特殊的優惠。
  9. 若在角色名稱上方未顯示聯盟公會徽章，只要打開「遊戲選項」中的UI分頁，勾選 ” 查看角色名稱 ( 聯盟公會會員 ) ” 選項，就會顯示左方的聯盟公會徽章囉！
  10. 功能說明：
    (1)點選快捷鍵(G)後，點擊下方分頁「聯盟」即可查看聯盟相關訊息。
    (2)在聯盟訊息中可查看聯盟公會成員列表。
    (3)點擊各個聯盟公會會長前的「+」按鈕，可開啟/隱藏該公會成員名單。
    (4)若聯盟會長踢出所有聯盟會員，則該聯盟會自動解散。
    (5)若聯盟中的任一公會佔有城堡，同聯盟公會成員向NPC購買物品可享有免稅優惠。

一.公會戰爭：

戰爭系統是公會間的合法性戰鬥，有以下幾點規則。
  1. 最多可以有5個公會進行戰爭。宣戰時輸入[/宣佈戰爭 (公會名稱)]。
  2. 停戰要滿足以下下幾個條件。
    (1)宣佈戰爭一周以後。
    (2)敵方的公會會長死亡三次。
    (3)敵方的公會成員死亡30次。
    (4)一方公會請求停戰或者投降。
  3. 停戰需輸入[/取消宣戰]。 或是在戰爭介面選擇要求停戰按鈕
  4. 投降需輸入[/投降 公會名稱]。 或是在戰爭介面選擇投降按鈕

戰爭中，城鎮NPC會參與嗎?

  1. 戰爭中，衛兵是不參與的。
  2. 戰爭中，與敵方公會成員進行PK不會使善惡值下降。
  3. 戰爭中，正當防衛規則將不會成立。
  4. 按G鍵選擇下端戰爭標籤就會出現戰爭視窗。

二.攻城戰：

1.如何發動攻城戰?

攻城需要由公會會長宣告，於「查斯泰爾」或「大墓地」城堡外地區輸入『/宣戰』

2.發動攻城的公會需要有什麼條件?

(1)公會會長需要達到30等級或以上
(2)公會會員數量必須達10人或以上
(3)沒有任何公會成員於任何城堡範圍內
(4)公會成員狩獵點數1000以上超過10人

三.《戰亂之聲》城堡分佈：

  1. 查斯泰爾地圖的「查斯泰爾城」

  2. 大墓地地圖的「拉姆茲城」

  3. 特羅圖斯地圖的「特羅圖斯城」

《戰亂之聲》現時有多少座城堡?

現時遊戲內有兩座城堡，分別是位於查斯泰爾地圖的「查斯泰爾城」及位於大墓地地圖的「拉吉姆基城」。

1.如何攻佔城堡?

在城堡外圍會有城門及守衛，攻破城門後便可入內對【守護塔】進行攻擊，成功破壞【守護塔】的公會會長，即代表已佔領該城堡。
倘若系統攻城時間尚未完結的話，現階段城主必須要守衛直至攻城時間完結為止。如中途被其他公會成功破壞【守護塔】的話，城堡擁有者將會轉變為最後破壞【守護塔】的公會。

搜索公會資訊

在村莊的公會佈告欄那裡可以查看所有公會的資料。用途如下：
  1. 申請加入公會
  2. 查看各公會詳細資訊
  3. 查看聯盟資訊
  4. 查看公會狩獵點數等等
  5. 每日00:00系統會更新

一.搜索與使用方式：

把鼠標移動到圖像上即可查看詳細內容


二.加入公會：

  1. 在欲加入公會的會長身上按「shift+左鍵」，點選「申請加入公會」按鈕即可。
  2. 在各主要村莊設置有「公會佈告欄」，尚未加入公會的玩家，透過公會佈告欄可搜索遊戲中所有的公會資訊。選擇欲加入的公會，並點選視窗的右下方「等待加入」按鈕後即完成申請程序。
