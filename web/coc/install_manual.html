安裝說明

電腦配備需求

項目 基本配備 建議配備
作業系統 WindowsXP、WindowsVISTA、Windows7、Windows8
CPU Pentium4 2.0Ghz 或 同等CPU Core E8400 3.0Ghz 或更高CPU
記憶體 4GB 8GB以上
硬碟空間 2GB 4GB
顯示卡 128M顯示記憶體
GeForce 6600或以上級別顯卡		 1G顯示記憶體
GeForce 9800GT或以上級別顯卡
音效卡 Windows 標準相容音效卡
網路寬頻 ADSL、 VDSL、 Cable同等級或以上速率
操作配備 滑鼠、鍵盤

顯卡驅動

  1. 遊戲安裝與執行時建議關閉防火牆及其他應用程式。
  2. 請確認電腦顯示卡的更新與驅動完全。
  3. 遊戲必須在DirectX 9.0c 以上版本環境下進行。若您電腦的DirectX版本較低，安裝遊戲主程式時，會直接協助玩家進行DirectX 9.0c安裝
  4. 重新安裝遊戲程式時，請確認是否完整移除舊遊戲主程式 ( 含資料夾 )
  5. 如以上資訊無法排除您安裝遊戲的問題，請多加利用官方LINE小天使或線上客服中心。
