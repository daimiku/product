FAQ

常見問題

Q：如何避免遊戲帳號遭盜用？

A：
  1. 官網下載防盜OTP APP程式，利用行動裝置提高帳號安全性。
  2. 不要隨意開啟不明網頁或下載不明檔案。
  3. 不使用外掛程式。
  4. 不將帳號密碼提供給他人。
  5. 定期進行電腦掃毒作業
  6. 公共場合登入遊戲時，留意身旁是否有可疑人士。
  7. 要暫時離開請先登出遊戲。
  8. 密碼請勿設定太過簡單，如：12345。
  9. 定時更換遊戲密碼與二次密碼。

Q：如何避免遊戲詐騙？

A：
  1. 避免與現金(或其他有價實體物品)有關的交易
  2. 謊稱中獎回覆帳號密碼即可領取
  3. 冒名詢問帳號密碼
  4. 勿開啟任何來路不明之郵件、檔案避免中毒
  5. 不採信任何免費獲得道具獎勵、現金等詐騙言論
  6. 保持良好的遊戲行為，不使用外掛或輔助程式等等，以上行為皆可能被種植木馬

Q：如果帳號遭到盜用的情況要如何處理？

A：

※有使用安全機制的情況※

  1. 請立即將電腦進行掃毒，或進行電腦重灌，確認安全後盡速變更密碼。
  2. 發現帳號遭重覆登入時，避免開通防盜OTP再次登入，請先至會員中心修改遊戲密碼，並立即通報客服中心。
  3. 請於24小時內傳真【會員服務申請表】至客服中心，並詳述遭盜用時間與情形，以便客服中心協助確認。
  4. 申請【帳號證明暨遊戲歷程調閱表】至轄區警察局報案。並將報案三聯單與相關文件申請盜用處理，以郵寄方式提供給客服中心。

※未使用安全機制的情況※

  1. 請立即將電腦進行掃毒，或進行電腦重灌，確認安全後盡速變更您的密碼。
  2. 立即通報客服中心，並於24小時內傳真【會員服務申請表】至客服中心，並詳述遭盜用時間與情形，以便客服中心協助確認。
  3. 申請【帳號證明暨遊 戲歷程調閱表】至轄區警察局報案。
  4. 耐心等候司法單位偵辦。
•依照線上遊戲定型化契約，遊戲歷程紀錄僅保留30日，請務必於期限內進行申請，逾期恕不受理。
•若因未設置安全機制導致帳號遭盜用，後續僅可由配合司法單位偵辦
•為確保帳號安全，建議使用官網提供的防盜OTP安全APP服務。

Q：萬一點數轉移錯的帳號怎麼辦？

A：儲值程序為客戶端自由操作，請於儲值時慎選帳號並多加確認，另外遊戲內商城物品可交易，也可以利用交易方式購買所需的道具轉移。

Q：出現儲值失敗，該帳號密碼不存在。

A：如果您購買的儲值點數為感熱紙張，請您在輸入儲值密碼時，注意儲值密碼只會有數字「0」和「1」，不會有英文「O」和「I」。若您購買的為實體卡片，請您在輸入儲值密碼時，注意儲值密碼字體較小為數字。

Q：忘記密碼怎麼辦？

A：可於登入會員頁點選『忘記密碼』，輸入遊戲帳號及驗證碼按下送出，系統將寄發密碼重置信至註冊填寫的信箱，請前往電子信箱收信並點選連結網址重置您的密碼。

Q：登入遊戲時一直出現帳號名稱或密碼錯誤？

A：如果您在登入的時候出現了"帳號名稱或密碼錯誤"這樣的訊息，請先試著再重新輸入一次，並確認大小寫區分問題，如果確認輸入沒有錯誤後還是出現同樣的訊息，建議您可以利用忘記密碼的處理流程進行密碼重置。

Q：申請會員帳號時E-mail填錯，造成現在收不到新的密碼信怎麼辦？

A：為維護每位玩家的個人資料安全，會員資料中的電子信箱是無法於官網直接修改的，需請您與客服中心聯繫。

Q：若申請了『忘記密碼』卻遲遲未收到該系統寄發的重置信？

A：若您的信箱為免費信箱，有可能會因垃圾信件阻擋功能造成您無法收到系統寄發的信件，建議建議玩家使用yahoo、hotmail、gmail等流通率較高的信箱。

Q：我的帳號被停權了，為什麼？

A：若您登入遊戲時發現自己的帳號被停權，您可與客服中心聯繫做進一步的詢問。

Q：請問我是否可以要求鎖定自已的帳號？

A：可以的。只要您為帳號所有人，皆可以利用傳真、親訪及郵寄的方式提供個人身分證明文件要求鎖定名下任一帳號。未來待您要解除帳號自我鎖定時，方式亦同。

Q：請問要如何修改『聯絡電話』？

A：請在登入會員後於會員專區『修改會員資料』中直接修改即可。

Q：如果註冊時填寫的身份證字號不是我本人的，請問可以修改嗎？

A：身分證字號在帳號註冊成功後便無法自行修改，若您有相關需求，可來信客服中心說明修改的事由，將由專人為您說明。

Q：如何關閉 Windows 防火牆？

A：雖然我們的遊戲主程式在安裝時會自動在您的 Windows 防火牆中註冊為安全軟體，某些狀況下您必須暫時關閉 Windows 防火牆才能夠連上遊戲伺服器。如果您確定伺服器狀況正常，但卻無法進行更新或登入遊戲，您可以嘗試暫時關閉 Windows 防火牆。以下是如何在 Windows XP、Windows Vista 和 Windows 7 中關閉 Windows 防火牆的教學：

【在 Windows XP 下關閉 Windows 防火牆】

  1. 點選開始功能表
  2. 點選控制台
  3. 點選安全性中心
  4. 點選 Windows 防火牆
  5. 點選關閉
  6. 點選「進階」分頁
  7. 取消勾選您的網路連線
  8. 點選「確定」
  9. 關閉安全性中心
  10. 略過安全性警告
  11. 按下「Windows 鍵」 + R
  12. 在跳出的「執行」視窗內輸入「services.msc」並按下確定
  13. 捲動列表，選擇 Windows 防火牆
  14. 在視窗的左上角點選「停止」
  15. 關閉所有視窗
  16. Windows 防火牆已經關閉，請執行遊戲主程式

【在 Windows Vista 或 Windows 7 下關閉 Windows 防火牆】

  1. 點選開始功能表
  2. 點選控制台
  3. 點選系統安全性
  4. 點選「Windows 防火牆」或「Windows Firewall」
  5. 點選進階設定
  6. 在中間的「總覽」區塊中點選「Windows 防火牆內容」
  7. 在「網域設定檔」、「私人設定檔」、「公用設定檔」三個分頁中都作以下的動作：
    (1)將防火牆狀態設置為「關閉」
    (2)點選「受保護的網路連線」旁邊的「自訂」按鈕
    (3)取消勾選您的網路連線，並點選確定，回到剛剛的視窗
    (4)點選確定
  8. 關閉所有視窗
  9. 略過安全性警告
  10. 按下「Windows 鍵」 + R
  11. 在跳出的「執行」視窗內輸入「services.msc」並按下確定
  12. 點選「Windows 防火牆」或「Windows Firewall」
  13. 在視窗的左上角點選「停止」
  14. 關閉所有視窗
  15. Windows 防火牆已經關閉，請執行遊戲主程式
